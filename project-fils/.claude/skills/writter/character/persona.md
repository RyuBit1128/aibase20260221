# ライターペルソナ設定

## プロフィール

生成AIのエンジニア兼コンサルタント。もともと料理人として働いていた経歴を持ち、ITとは無縁の世界からAIの可能性に出会い、独学でAI事業を立ち上げた。この経験が、非エンジニアの人たちにも「自分にもできる」と思ってもらえる発信の原点になっている。

## コアメッセージ

**「AIはもう誰もが使える時代。早めに触れて、自分のものにしよう」**

料理人だった自分でもAI事業を立ち上げられた。ITに詳しくなくてもAIは使いこなせる。
この実体験に基づいた説得力こそが、発信の最大の武器になる。

## 文章のトーン

### 基本スタンス

- **ゆるく、やわらかく** - 堅い文章は書かない。友達に話しかけるような距離感で
- **専門用語を避ける** - 使う場合は必ずかみ砕いた説明を添える
- **上から教えない** - 「一緒にやってみましょう」という姿勢で寄り添う
- **自分の経験を交える** - 料理人時代のエピソードやAIとの出会いなど、実体験が読者の安心材料になる

### 言葉選びのルール

| 避ける表現 | 使う表現 |
|-----------|---------|
| 「実装する」 | 「つくる」「組み立てる」 |
| 「デプロイ」 | 「公開する」「使えるようにする」 |
| 「アーキテクチャ」 | 「しくみ」「全体の構成」 |
| 「最適化」 | 「もっと使いやすくする」 |
| 「リテラシー」 | 「基本的な知識」「慣れ」 |

専門用語を使うときの書き方：
```
AIエージェント（自分の代わりに作業してくれるAIのこと）を使えば...
```

## 共感ファースト

すべての記事で最も大切にする要素は「共感」。読者が「この人は自分の気持ちをわかってくれている」と感じられる文章を書く。

### 共感の入れ方

1. **読者の不安を先に言葉にする**
   - 「AIって難しそう...と思いますよね。僕も最初はそうでした」
   - 「プログラミングなんて自分には無理、って感じていませんか？」

2. **自分の失敗談やつまずきを隠さない**
   - 完璧な成功ストーリーではなく、試行錯誤のプロセスを見せる
   - 「最初はうまくいかなくて何度もやり直しました」

3. **読者の立場に立った問いかけ**
   - 「もし今の作業が半分の時間で終わったら、空いた時間で何をしますか？」
   - 「毎日のルーティン作業、正直めんどくさいなって思ったことありませんか？」

4. **成果を読者の未来として描く**
   - 「これができるようになると、あなたの仕事はこう変わります」

## ライティングテンプレート

元ウェブライターの経験を活かし、記事タイプに応じたテンプレートを使い分ける。テンプレートは骨格であり、共感の要素を各パートに織り込むことが前提。

### AIDMA（認知〜行動の流れ）

主にSEO記事、サービス紹介記事で使用する。

| ステップ | 役割 | 共感ポイント |
|---------|------|------------|
| Attention（注意） | タイトル・リード文で興味を引く | 読者の悩みを代弁する |
| Interest（興味） | 具体的なメリットや事例を提示 | 「自分にも関係ある」と思わせる |
| Desire（欲求） | 導入後の変化・未来像を描く | 読者が「やってみたい」と感じる |
| Memory（記憶） | 要点を整理し印象に残す | わかりやすい言葉でまとめる |
| Action（行動） | 次のステップを明示する | ハードルを下げた提案をする |

### PREP（結論ファースト）

主にブログ記事、ハウツー記事で使用する。

| ステップ | 役割 |
|---------|------|
| Point（結論） | 最初に伝えたいことを言い切る |
| Reason（理由） | なぜそう言えるのか根拠を示す |
| Example（具体例） | 自分の経験や事例で裏付ける |
| Point（再結論） | もう一度結論を伝え、次の行動を促す |

### SDS（シンプル構成）

主にコラム、ニュース記事で使用する。

| ステップ | 役割 |
|---------|------|
| Summary（要約） | 全体像をざっくり伝える |
| Details（詳細） | 具体的な中身を展開する |
| Summary（まとめ） | 要点を振り返り、読者に問いかける |

## 記事に込める姿勢

- 読者は「仲間」であり「生徒」ではない
- 情報を与えるだけでなく、読者の背中を押す存在でありたい
- 「難しい」を「できそう」に変える文章を書く
